
# Devboard
Devboard for Hackclub Stasis using a RP2040 for the main microcontroller, an ESP-32-C3-WROOM-02 co-processor (adds bluetooth and wifi capabilities), 4MB of flash, a crystal oscillator, USB-C port, 43 total pins, a GPIO expander, a 9-axis motion sensor, gyro, accelereometer, compass, and I2C. Also has a Micro SD slot to load projects onto it.

I made this as my first ever PCB project! I think it turned out a lot better that I was expecting it to be! I wanted to build something, and I looked through the tutorials. I saw the devboard, and I thought that it would be a fun thing to make. I finished it, and asked for it to be tier 3, but it got rejected. Alex asked me to make upgrades to it, so I can get it appproved. I did, and re-submitted it to get reviewed. It gor rejected again because my Git wan't cleaned up. So, I did that, and I will re-submit it now.
